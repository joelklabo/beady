## [0.1.0] - 2025-12-17

### Added
- Issues explorer view for browsing Beads issues inside VS Code.
- Commands to create, open, and refresh issues from the Command Palette.
- Dependency visualization (dependency tree + graph view).
- Activity feed view for recent issue changes.
- Worktree-aware workflows (worktree guard support and multi-root resolution).
- Built-in chat participants for agent workflows (`@task-creator`, `@task-worker`).
